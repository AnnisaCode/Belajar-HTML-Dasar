Belajar Menggunakan Link

Menggunakan Link, kita dapat berpindah dengan cara meng-klik link atau tautan tersebut. Link tidak selalu dalam bentuk teks, bisa juga dari gambar.

Daftar Tag Link

  1. Tag <a>
    1. Isi konten tag ini adalah isi dari tampilan link, bisa berupa teks atau lainnya.
    2. Tag ini memiliki atribut href, yang berisi lokasi tujuan dari link tersebut.
    3. Tag <a> juga memiliki atribut target, yang digunakan sebagai target jendela Web Browser. Kita bisa menggunakan nilai seperti di bawah ini:
      1. target="_self", artinya halaman akan ditampilkan di halaman yang sama. Ini adalah bawaan default.
      2. target="_blank", artinya halaman ini akan ditampilkan di jendela baru di Browser.
    4. Tag <a> juga memiliki atribut title, untuk menuliskan judul yang muncul ketika mouse berada di atas link tersebut.

Berikut adalah contoh link menggunakan tag <a>:

  • Contoh Link ke Example.com
  • Buka Example.com di Tab Baru
  • Kembali ke Halaman Utama

Absolute Link dan Relative Link

Dalam pembuatan link, kita dapat menggunakan dua jenis tautan:

  1. Absolute Link

    2. Absolute link adalah tautan yang mencakup URL lengkap, termasuk protokol (seperti http atau https), nama domain, dan jalur ke file. Link ini akan selalu mengarah ke lokasi yang sama di internet, terlepas dari halaman tempat link tersebut diletakkan.

    Contoh absolute link:

    • Example.com - About Us
  2. Relative Link

    3. Relative link adalah tautan yang hanya mencakup jalur relatif terhadap lokasi file saat ini. Link ini biasanya digunakan untuk menghubungkan halaman-halaman di dalam situs yang sama. Relative link bergantung pada struktur direktori situs dan lokasi file saat ini.

    Contoh relative link:

    • Hello World Page (relative link di folder yang sama)
    • Home (relative link di folder yang berbeda)
