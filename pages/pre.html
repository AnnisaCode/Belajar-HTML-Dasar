Menggunakan Pre

Dengan menggunakan tag pre, kita dapat menampilkan baris apa adanya sesuai apa yang kita code di html:

        public static void main(String[] args) {
            System.out.println("Hello World!");
        }

Jika kita menggunakan tag p, maka hasil nya browser akan menampilkan code diatas menjadi 1 baris saja karena dari browser nya sudah menghapus enter dan spasi dari html nya:

public static void main(String[] args) { System.out.println("Hello World!"); }
